CarPlay Powered by Blackberry➞

March 4th, 2014

BGR:

BlackBerry’s QNX platform is the base on which the BlackBerry 10 operating system is built, but it’s also much more than that. Before it was morphed into smartphone software, QNX was an in-car operating system that powered infotainment systems in a wide range of vehicles. It still is, and as N4BB confirmed on Tuesday, it’s also the foundation for Apple’s new CarPlay product.

Amazing that the value for QNX may not come from Blackberry mobile devices, but from being the go to in-car operating system. Add this onto news that Ford Sync is planning to drop Microsoft for QNX, and it looks like Blackberry is building itself a nice little niche, though I'm not sure how profitable it is.

I never would have believed this a year or so ago, but is it possible that Blackberry could become an acquisition target for Apple?
